

## How to use the stub

```html
<div id="otter-preview"></div>
<script src="../../js/otter-input-bar.js"></script>
<script>
  new OtterLiteBar('otter-preview');
</script>
```

The constructor simply paints a CTA card. There are no runtime options or callbacks in this lite build.

Deploy a site on [vibeotter.com](https://vibeotter.com). Customers there get:

- Animated otter assistant with conversational memory
- File + link understanding
- Claude 4.5 Sonnet orchestration
- Live preview feed
- Shopify-aware prompts and actions
